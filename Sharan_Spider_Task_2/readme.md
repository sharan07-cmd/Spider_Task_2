# Octo-Shell

A custom, interactive OS shell natively built in C. This project
features process management, a dual-terminal encrypted file
transmission protocol (The Radio), and a kernel-level Intrusion
Detection System (Tripwire).


You can watch the full 2-minute demonstration of the dual-terminal file transfer and the Tripwire alert system here: [Watch Demo Video Link](https://drive.google.com/file/d/1Pxblhx4zqZ19e379kP7X5_uTq9_o--nl/view?usp=drive_link)
