\# Octo-Shell

A custom, interactive OS shell natively built in C. This project
features process management, an ephemeral dual-terminal encrypted file
transmission protocol (The Radio), and a kernel-level Intrusion
Detection System (Tripwire).

\## Compilation Instructions

This project includes a \`Makefile\` for seamless compilation. To clean
up old binaries and compile the entire project from scratch, open your
terminal in the project directory and execute:

\`\`\`bash

make clean && make
